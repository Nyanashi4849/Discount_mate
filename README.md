# DiscountMate API

Smart Grocery Discount Prediction API

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Docker
- Jenkins

## Setup

Install dependencies

npm install

Run server

npm run dev

## Docker

docker-compose up --build

## API Endpoints

GET /products  
POST /products  
POST /price/update  
GET /price/history/:product_id  
POST /alerts  
GET /alerts
